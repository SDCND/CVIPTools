Name

CLAHE_Grid

performs contrast limited adaptive histogram equalization with a non-overlapping grid

Function Prototype

Parameters

Type

clippercent

double

desired blocksize

desired clip percentage

mb

int

RGB band on which to calculate histogram (0,1,2)

Description

 
This function performs a contrast limited adaptive histogram equalization algorithm. 
The algorithm calculates the CDF for each grid section based on the blocksize. Pixels are then reassigned values using the CDF. Images must be a byte value.
