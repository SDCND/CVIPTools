 Compare two images:

It compares the current image with the second image by subtraction or a logical XOR

First, select the method of comparison with mouse button 1.

The current image is automatically set, and can be changed by clicking mouse button 1 on an image in the main window’s image queue. The second image can be selected from the pop-up menu.

Objective Error Metrics:

It compares the current image with the second image by implementing the

RMS error, Peak SNR, XOR error, Dice coefficient, Jaccard index, or Overlap coefficient.

First, select the method of comparison with mouse button 1.

The current image is automatically set, and can be changed by clicking mouse button 1 on an image in the main window’s image queue. The second image can be selected from the pop-up menu.
After making your selections, press ‘Apply’ to see the difference within the results textbox.

 

Pratt Merit:

This function calculates the Pratt’s Figure of Merit of the image. The Pratt FOM, is defined as follows:

IN = the maximum of II and IF

II = the number of ideal edge points in the image

IF = the number of edge points found by the edge detector

α = a scaling constant that can be adjusted to adjust the penalty for offset edges

di = the distance of a found edge point to, an ideal edge point

For this metric, F will be 1 for a perfect edge. Normalizing to the maximum of the ideal and found edge points guarantees a penalty for smeared edges or missing edge points. In general, this metric assigns a better rating to smeared edges than to offset or missing edges. This is done because techniques exist to thin smeared edges, but it is difficult to determine when an edge is missed. The distance, d, can be defined in more than one way and typically depends on the connectivity definition used. To use, select the ideal edge image and the edge detection image from the pop-up menu. Enter or select the scale factor from the pop-up menu. The scaling factor is the scaling constant that can be adjusted to adjust the penalty for offset edges. Press the ‘Apply’ button to perform the operation. The Pratt’s Figure of Merit appears in the ‘Result’ box. For multi-band (color) images, Pratt FOM returns the value for each RGB band separately.
