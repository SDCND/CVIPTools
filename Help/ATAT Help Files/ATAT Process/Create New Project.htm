Creating a New Project:

1.      Once the Main ATAT GUI is open, click the File option on the top left corner.

2.      Choose New Project from the drop down.

3.      Give the project a name and location. A folder will be created inside this location with the project name specified. All result images, error measurements, and configuration files will be saved in this location. The browse option can be used to look for a folder location. Click OK to create the new project.

4.      After this, the Images Form will open. The ATAT application does not save after each change. Make sure to click “yes” when closing the project, or the configuration may be lost or converted to a previous session. If an “Unexpected Exception” error is found when clicking “yes”, it means the current configuration has already been saved.
