Images Interface:

 

Adding Test Images and Ideal Images:

            There are two types of images used in the ATAT application. The first is referred to as images, or test images. These will be used in the algorithm development section of experimentation. To add a test image, the button “Add Image” should be selected. This will bring up a file menu where you can choose what test images to add. If the test images are successfully retrieved, they will show up in the image table. The test image name and path will be displayed in the first two columns. The amount of test images will also change the Total Images count.

            The other type of image is the ideal image. These are used for the comparison section of the process. Once the test images are run through the different algorithms, their resulting images will be compared to the ideal counterpart. Because of this, the ideal images are based on the test images. To add an ideal image, the “Add Ideal Image” button should be selected. This will bring up a file menu and the ideal images can be selected. These images are required to have the same name as the test images that they will be compared with. When running tests, two different folders should be used for each type of image. If the ideal image has found a test match and can be successfully read, it will be added to the table next to its test image. These will fill in the second two columns.

            Images can also be added with a batch method using a folder containing test images and a sub-folder, titled Masks, that holds the ideal images. Use the “Add Folder” button to select the batch folder.

Table Commands/Features:

            Images will be added using the “Add Image” and “Add Ideal Image” buttons. If these are successful, the table will be filled with the specified images. An image can be selected by clicking on the row it appears in. There are four options for manipulating the table. Once a row is selected, the image or ideal image can be selected via that first and second delete buttons. All images or ideal images can be deleted via the ALL delete buttons.

            The table is broken up into 6 columns. The first four give location and name data of the test and ideal images chosen. The final two columns will display the row and column sizes of the images.
