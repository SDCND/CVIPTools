Requirements:  A computer with at least 100GB of hard drive (depends on the amount of images used in the experiment) and at least 2GB of memory. For more information see Chapter 12 in the textbook: Digital Image Processing & Analysis: Human and Computer Vision Applications with CVIPtools, Second Edition, Scott E Umbaugh,  CRC Press, 2020

 

The GUI:

            - The Main Interface

            - The Images Interface

            - The Algorithm Interface

            - The Test and Compare Interface

 

The Main Interface:

            The interface that pops up once the CVIP - ATAT is started is the Main Interface. To create a project the user should select File from the menu bar above. From there the user should select either New Project or Open Project if a project has already been created.

 

The Image Interface:

            The image interface is the first window that comes up once the project has been created or opened. Here the user can insert test images along with their corresponding "ideal" images which can later on be used during the image comparison process. Note: The name of the ideal images must be the same as the original images otherwise the ATAT will not add any images to the Ideal Image column.

 

The Algorithm Interface:

            The algorithm interface is where the user can insert the processes that they would like to use for their algorithm. They must select the desired range of parameters for every process. Note: Some processes have parameters that should be fixed to one value only. Make sure to be aware of that otherwise the CVIP - ATAT will fail to operate. The way to fix a value is to enter the same value for the lower and upper value text boxes. The increment does not have to be the same value as the lower and upper values.

 

The Test and Compare Interface:

            The test and compare interface is where the experiments, defined by the Image and Algorithm interfaces, can be run. The type of experiment executed can be chosen in the testing section. If a comparison experiment is chosen, the Compare section can be changed to specify error metrics that will be used during the experiment. The current progress of the test will be monitored at the top of the interface, and the images created will be added to the table at the bottom of the interface. The comparison results will be found in an excel sheet located in the same folder as the project file.
